## F. 移植順序建議（降低風險）

1. **先立骨架**：Tauri + React + Vite 跑起來、SQLite 能存讀。
2. **物件樹模型 + 渲染器 + 一個 preset**：能把手寫的物件樹 JSON 畫出來。
3. **編輯器最小可用**：選取、拖曳、控點縮放、文字編輯。
4. **接 AI pipeline**：parse → 翻譯 → 大綱 → 選 preset → 翻譯成物件樹。
5. **主題與動效**：套 23 主題、進場/轉場。
6. **匯出**：HTML → PDF → PNG → PPTX（依難度遞增）。
7. **設定**：多 provider API Key + 動態模型清單。
8. **打包**：Tauri 產出 `.exe`，驗證雙擊即開。
