# CourseSlideGenerator（教學簡報產生器）

一個以 `React（前端函式庫）` + `Vite（前端建置工具）` + `Tauri（桌面應用框架）` 製作的教學簡報編輯器，支援：

- 文章匯入後由 `LLM（大型語言模型）` 生成簡報
- 主題切換（含 `Reveal.js（簡報框架）` 風格）
- 圖片與音樂資產管理
- 瀏覽器模式與桌面模式

## 環境需求

- `Node.js（JavaScript 執行環境）` 18 以上
- `npm（套件管理器）` 9 以上
- 若要封裝桌面執行檔，需安裝：
  - `Rust（系統程式語言工具鏈）`
  - `Microsoft C++ Build Tools（Windows 編譯工具）`
  - `WebView2（Windows 網頁執行元件）`（通常 Win10/Win11 已內建）

## 安裝相依套件

```bash
npm install
```

## 在瀏覽器啟動伺服器

```bash
npm run dev
```

啟動後會看到本機網址（通常是 `http://localhost:5173` 或 `http://localhost:5180`）。  
將網址貼到瀏覽器開啟即可使用。

## 開發桌面版（Tauri 即時模式）

```bash
npm run tauri:dev
```

此指令會先啟動前端開發伺服器，再開啟桌面視窗。

## 建置前端靜態檔

```bash
npm run build
```

輸出目錄為 `dist`。

## 封裝成單一執行檔（Windows）

1. 先確認已安裝 `Rust（系統程式語言工具鏈）` 與 Windows 編譯工具
2. 執行：

```bash
npm run tauri:build
```

3. 產物位置（預設）：
   - 安裝程式（`MSI（Windows 安裝格式）`）通常在 `src-tauri/target/release/bundle/msi`
   - 單一可攜執行檔（若有啟用）通常在 `src-tauri/target/release`

> 若你想要「完全單檔免安裝」版本，可後續調整 `Tauri（桌面應用框架）` 打包目標與 `bundle（封裝）` 設定。

## 常用指令

- `npm run dev`：啟動瀏覽器開發伺服器
- `npm run build`：建置前端
- `npm run preview`：預覽建置結果
- `npm run tauri:dev`：啟動桌面開發模式
- `npm run tauri:build`：打包桌面執行檔

## 品質檢查與設定

- 在上方工具列點「設定」，可調整「AI 覆蓋率門檻」滑桿（預設 90%）
- 生成完成後，系統會顯示：
  - 大綱覆蓋率（補頁前/補頁後）
  - 自動補頁張數
  - 內容品質檢查（事實一致性、過密頁面拆分）
- 若覆蓋率低於你設定的門檻，系統會自動再進行強制補頁
- 你可在 `AI 生成` 面板按「匯出報告 JSON」下載檢查結果

## 進階編輯功能

- 一鍵重新排版：工具列「重新排版」可在不改文字內容下，重新整理目前頁面版面
- 批次頁面操作：左側面板可多選頁面後，一次套用主題與轉場
- 講者備忘稿：在「匯出簡報」可直接輸出 `Markdown（標記語言）` 講稿
- 版本快照：工具列「快照」可建立命名快照、還原與刪除
- 匯出面板已整合「品質報告 JSON」下載
- Reveal.js 主題已擴充至最新版全套（含 `dracula（德古拉）`），並加入更細緻的主題樣式變數映射
